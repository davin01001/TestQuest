Приложение может производить следующие манипуляции с данными:
	- Соединяться с сервером
	- выводить список баз и несистемных таблиц
	- добавлять/удалять таблицы
	- добавлять/удалять поля в выбранную таблицу
Приложение использует методы доступа к данным SqlClient
Приложению необходимо для работы:
	- установленный SQL Server на сервере
	- .NET Framework 2.0 на клиенте
